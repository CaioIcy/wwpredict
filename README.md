# wwpredict

Tries to predict the outcome of your [Werewolf for Telegram](https://github.com/GreyWolfDev/Werewolf) game.

### Usage

Configure main.py and run:

```sh
$ python main.py
```

### Dependencies

```sh
pip3 install flake8
pip3 install nose
```

### Tests

```sh
nosetests -sv ./tests/*.py
```
